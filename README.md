# Hotel Booking Form
The Hotel Booking Form is a simple and user-friendly web application designed to collect hotel reservation details from users. This project demonstrates form design, input validation, and basic frontend development concepts.

## Project Overview
A responsive hotel booking form with date validation and dynamic interaction.

## Features
- Check-in & Check-out validation
- No past date booking
- Responsive design
- Dynamic error messages

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download the project
2. Open index.html in browser
